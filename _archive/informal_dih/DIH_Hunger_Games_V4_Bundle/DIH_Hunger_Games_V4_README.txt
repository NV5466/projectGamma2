DIH Hunger Games V4 Bundle
===========================

Contents
--------
1. dih_hunger_games_v4_prototype.py
   Standalone prototype implementing the hierarchical scoring architecture.

2. dih_hunger_games_v4_summary.csv
   Final synthetic-test scores and classifications.

3. dih_hunger_games_v4_layer_evidence.csv
   Aggregated evidence used by the relay, sensor, and compound models.

4. dih_hunger_games_v4_capture_features.csv
   Per-capture waveform-derived features.

5. dih_hunger_games_v4_scores.png
   Score comparison chart.

Important
---------
This version was validated against synthetic waveforms. It is an architecture
prototype, not yet a validated hardware diagnostic model. Real capture testing
and parameter calibration are still required.
