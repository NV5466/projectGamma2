# ElectroStat Parts Folder

This folder gives each milestone its own place so the project can be browsed by part number.

- `part-1-analog-report-spine`
- `part-2-role-aware-routing`
- `part-3-structured-display-reports`
- `part-4-first-class-analysis-windows`

The live source remains in `raildiag/`. These folders are milestone notes and navigation anchors, not duplicate source snapshots.

Each part folder also has local `artifacts/` output from representative runs, including plots where that milestone generated them.
